Instructions to run the code successfully:

1. Download the whole Mini Project folder on your local device.
2. Kindly when using Visual Studio Code to run the Hangman Game, 
   open the whole Mini Project folder which includes the text files and the code.
3. Then run the code as usual.

NOTE: It is necessary to open the folder first in VSC for the code to run properly,
	otherwise the program won't be able to detect the text files to load the words.
